# Dependencies

This file describes the project dependencies and the primary reason for their use.

## MarkedJS

Ref: [MarkedJS](https://github.com/markedjs/marked)
This library is used to parse Markdown into HTML.

## DOMPurify

Ref: [DOMPurify](https://github.com/cure53/DOMPurify)
This library is used to protect against XSS attacks in HTML.
